XTrans.zip - does not required installation, but you must have VCRT8 and MFC8 v8.0.50727.4053 installed. (in the \Windows\WinSxS folder). 
If XTrans does not start, try to install the Microsoft Visual C++ 2005 Redistributable Package (x86) - vcredist_x86.exe
Also it can be downloaded from the Microsoft site:
http://www.microsoft.com/downloads/en/details.aspx?familyid=766a6af7-ec73-40ff-b072-9112bab119c2&displaylang=en
